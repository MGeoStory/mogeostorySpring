相較台北、新北或新竹市等地方而言，便利商店在離島(澎湖、金門、馬祖)、彰化和屏東等地，可能是採買的好去處，而不僅僅只是買個方便的地方。

透過檢視地圖中2013年至2016年的便利商店消費平均客單價資料，可以發現幾件很不一樣的事情：

  • 便利商店的消費習慣可能存在著區域差異：離島(澎湖、金門、馬祖)、彰化和屏東等地，通常是消費金額最高的前三名；而台北市 、新北市和新竹市等地，則多是消費金額最少的三個地區之一。如果納入其他類型賣場的空間位置與影響範圍，應該可以看到更多資訊。
  • 金門一直到2013年12月才有在便利商店：檢視數據之後才發現這件事情，而且金門的消費金額一直逐年上升，這點是因為品項與其他地區有所不同嗎？
  • 氣溫對消費金額的影響：無論南北，只要是偏冷的一月，都是平均消費金額最高的時候。另外在南部地區，七月的炎熱也讓便利商店的生意達到另一個高峰。
